# 快速启动指南

## 前端本地运行

### 方法一：Python（推荐，无需安装）
```bash
# Python 3
python -m http.server 8080

# 浏览器打开
# http://localhost:8080
```

### 方法二：Node.js
```bash
npx serve . -p 8080
# 或
npx http-server . -p 8080 -o
```

### 方法三：VS Code
安装 Live Server 插件，右键 index.html → Open with Live Server

### 最小API接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET  | `/api/tour-spot/detail/{id}` | 景点详情 |
| GET  | `/api/tour-spot/comment/list/{spotId}` | 景点评论 |
| POST | `/api/tour-spot/comment/add` | 提交景点评论 |
| GET  | `/api/folk/detail/{id}` | 民俗详情 |
| GET  | `/api/folk/comment/list/{folkId}` | 民俗评论 |
| POST | `/api/folk/comment/add` | 提交民俗评论 |

响应格式：`{ "code": 200, "msg": "success", "data": { ... } }`
