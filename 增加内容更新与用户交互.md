# 增加内容更新与用户交互 - 安多藏族文化模块说明

## 一、功能升级说明

### 1. 核心功能升级
- **数据驱动升级**：将原有的静态数据改为动态数据，通过API接口获取民俗文化和旅游景点信息
- **交互体验优化**：实现流畅的横向滚动卡片效果，支持鼠标拖拽和触摸滑动
- **响应式设计**：确保在不同设备上都能良好展示
- **视觉效果增强**：添加卡片悬停动画、按钮渐变效果等

### 2. 技术实现升级
- **数据管理**：实现从后端API获取数据，并在API失败时提供默认数据作为降级方案
- **状态管理**：优化Vue实例的状态管理，确保数据更新时的视图响应
- **性能优化**：使用Vue 3的响应式系统，减少不必要的重渲染

## 二、页面优化说明

### 1. 视觉设计优化
- **色彩方案**：采用藏族传统色彩搭配，以红色和金色为主色调，体现民族特色
- **排版布局**：合理的卡片间距和布局，确保视觉层次感
- **动画效果**：添加卡片悬停动画、按钮点击反馈等微交互
- **响应式布局**：适配不同屏幕尺寸，确保在手机、平板和桌面端都有良好表现

### 2. 用户体验优化
- **交互流畅度**：实现平滑的滚动效果和过渡动画
- **导航便捷性**：添加直观的箭头导航按钮和分页指示器
- **加载状态**：在数据加载过程中提供适当的加载提示
- **错误处理**：当API请求失败时，自动切换到默认数据，确保页面正常显示

## 三、需求分析

### 1. 前端交互需求
- **卡片滚动**：支持横向滚动查看更多内容，可通过箭头按钮或鼠标拖拽操作
- **点击交互**：点击"了解更多"按钮跳转到详情页，并传递相关参数
- **悬停效果**：卡片悬停时显示放大效果和阴影变化
- **响应式适配**：在不同设备上自动调整布局和交互方式

### 2. 页面需求
- **模块布局**：两个独立的内容模块，分别展示民俗文化和旅游景点
- **卡片设计**：每个卡片包含图片、标题、描述、标签和操作按钮
- **导航元素**：包含左右滚动箭头和分页指示器
- **数据展示**：以卡片形式展示数据，确保信息清晰易读

## 四、功能模块明细

### 1. 安多藏族民俗文化模块
- **模块ID**：`culture`
- **数据来源**：`/api/folk/list` API接口
- **数据结构**：
  ```javascript
  [
    {
      id: Number,        // 唯一标识符
      title: String,      // 民俗名称
      desc: String,       // 民俗描述
      img: String         // 图片路径
    }
  ]
  ```
- **交互功能**：
  - 横向滚动查看更多民俗项目
  - 点击"了解更多"按钮跳转到详情页
  - 卡片悬停效果

### 2. 藏地旅游+民俗模块
- **模块ID**：`tour`
- **数据来源**：`/api/tour-spot/list` API接口
- **数据结构**：
  ```javascript
  [
    {
      id: Number,        // 唯一标识符
      title: String,      // 景点名称
      desc: String,       // 景点描述
      img: String         // 图片路径
    }
  ]
  ```
- **交互功能**：
  - 横向滚动查看更多景点
  - 点击"查看详情"按钮跳转到详情页
  - 卡片悬停效果

## 五、技术栈

### 2. 核心技术点
- **Vue 3 Composition API**：使用Vue 3的响应式系统管理状态
- **CSS变量**：统一管理颜色、尺寸等样式变量
- **Flexbox布局**：实现灵活的卡片布局
- **Scroll Snap**：实现卡片的对齐滚动
- **Fetch API**：从后端获取数据
- **Error Handling**：实现API请求失败的降级处理

## 六、前端与后端的连接说明

### 1. API接口设计
- **民俗文化列表接口**：`/api/folk/list`
  - **请求方式**：GET
  - **响应格式**：
    ```json
    {
      "code": 200,
      "data": [
        {
          "id": 1,
          "title": "藏族情歌《拉伊》",
          "desc": "形式：山间对唱；习俗：青年恋爱交流",
          "img": "image/mn1.jpg"
        }
      ]
    }
    ```

- **旅游景点列表接口**：`/api/tour-spot/list`
  - **请求方式**：GET
  - **响应格式**：
    ```json
    {
      "code": 200,
      "data": [
        {
          "id": 1,
          "title": "塔尔寺",
          "desc": "青海·西宁｜藏传佛教圣地",
          "img": "image/nn1.jpg"
        }
      ]
    }
    ```

### 2. 数据加载流程
1. **初始化**：页面加载时，Vue实例初始化，folkList和tourList为空数组
2. **数据请求**：mounted钩子中调用loadFolkList和loadTourList方法
3. **API调用**：使用fetch API请求后端接口
4. **数据处理**：
   - 成功：将返回的数据赋值给对应的数组
   - 失败：使用默认数据作为降级方案
5. **视图更新**：数据更新后，Vue自动更新视图，展示卡片列表

### 3. 错误处理机制
- **网络错误**：当API请求失败时，捕获错误并使用默认数据
- **数据格式错误**：检查返回数据格式，确保数据结构正确
- **空数据处理**：当返回数据为空时，显示默认提示信息
